== HTML Formatting ==
Valid HTML markup is almost fully supported. See http://www.w3.org/MarkUp for more details.

== Links ==
  By default any valid link will be displayed as link automatically
  http://example.org

  To transfor something to link you should include text into double brackets
  <nowiki>[[Transform to Link]]</nowiki>
  [[Transform to Link]]

== Image Inclusions ==

By default any link to the image will include this image into the page

  <nowiki>http://acunote.com/images/logo_acunote_promo.gif</nowiki>
  http://acunote.com/images/logo_acunote_promo.gif

To render link to the image as link you should include it to double brackets (see [[#Links|Links]] section).<br/>
Images aslo can be included with HTML &lt;img&gt; (see [[#HTML Formatting|HTML Formatting]] section).
